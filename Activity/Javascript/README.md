## How to Run

### Node.js files (.js)
```bash
node basics/01_conversion_operators.js
node exp1/01_experiment1.js
# etc.
```

### HTML files (.html)
Open directly in any browser (double-click or drag into browser).

## Topics Covered

| Folder           | Topics |
|------------------|--------|
| basics           | Type conversion, arithmetic, strings, null/undefined, increment, comparison, Math.random |
| function_activity| Functions (all types, closures, HOF), String methods |
| array_activity   | split, splice, slice, toSpliced |
| dom_activity     | jQuery selectors, event handling, DOM manipulation |
| async_await      | Guess game, blocking vs non-blocking, real-life examples, fetch API |
| fetch_activity   | fetch line-by-line, async/await fetch, user names list, first 5 posts, fake promise |
| promises         | Callbacks vs Promises, Promise.all, Promise.race, Promise.allSettled |
| exp1             | Student info, even/odd, pass/fail, print 1–10, variable assignment, console |
| exp2             | Arrays/functions/objects, reverse number, palindrome, fibonacci, vowels, largest, duplicates, missing number, sum |
| exp3             | Arrow functions, switch case, truthy/falsy, ternary, loops, map & filter |
