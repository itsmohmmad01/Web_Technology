Data Binding is a technique to connect data between the Component (TypeScript) and the View (HTML).

There are 4 types of Data Binding in Angular:

1. INTERPOLATION (One-way binding: TS → HTML)
Used to display data from component to view.
Syntax:
{{ variableName }}

2. PROPERTY BINDING (One-way binding: TS → HTML)
Used to bind data to HTML element properties.
Syntax:
[property]="variableName"

3. EVENT BINDING (One-way binding: HTML → TS)
Used to handle events from the UI.
Syntax:
(event)="functionName()"

4. TWO-WAY BINDING (TS ↔ HTML)
Used for both input and output together.
Syntax:
[(ngModel)]="variableName"
